# Adaptive Question Difficulty System (Python OOP)

This project implements an adaptive study planner using Object-Oriented Programming (OOP).  
It adjusts difficulty and study time based on learner performance.

## Features
- OOP Classes: Learner, Subject, StudyPlan, ProgressTracker
- JSON Data Persistence
- Adaptive difficulty modification
- Menu-based CLI system

## How to Run
```
python main.py
```
